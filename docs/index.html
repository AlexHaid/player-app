Репозиторий учебного проекта разработки музыкального плеера.

Frontend: react, react-redux, react-saga, styled-components Backend: node.js (to be announced)

Локальный запуск проекта:

npm start

Позволяет запустить проект в режиме разработки. URL: [http://localhost:3000](http://localhost:3000

npm test

Запускает написанные к компонентам тесты.

npm run build

Запускает билд для деплоя на сервер. Скрипты собираются в один бандл и он дополнительно сжимается для большей производительности.

npm run eject

Note: this is a one-way operation. Once you eject, you can’t go back!

Позволяет изменить конфиги webpack, babel, eslint и т.д. Копирует все конфиги и зависимости, выдавая пользователю контроль над ними. Действие нельзя откатить.

По мере разработки этот файл будет обновляться и дополняться.
